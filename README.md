# 🌦️ Weather Intelligence Dashboard

A modern, responsive weather intelligence application built for the **PM Accelerator** AI Intern assessment. This tool provides real-time atmospheric data and travel insights for the modern explorer.

## 🚀 Key Features
- **Real-time Weather Data:** Integration with OpenWeatherMap API for live updates.
- **Geolocation Support:** One-click weather fetching based on the user's current coordinates.
- **Travel Intelligence:** AI-driven insights that translate raw data (wind, temp) into actionable travel advice.
- **Search History:** Persistent recent searches stored via browser `localStorage`.
- **Professional UI:** Built with Next.js, Tailwind CSS, and Lucide React icons.

## 🛠️ Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS v4
- **Icons:** Lucide React
- **State Management:** React Hooks (`useState`, `useEffect`)

🚀 To Run locally:

Clone the repo

Run npm install

Add your API Key to .env.local (check .env.example file)

Run npm run dev
